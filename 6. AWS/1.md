## Q. What is instance fleet in AWS?

### Answer:
-   It is nothing but combination of different kinds of EC2 Instances, both with respenct to the capacity and pricing models.

-   A fleet can launch multiple instance type across multiple AZ, using On-Demand Instances, Reserved Instances and Spot instances **purchasing option together.**

### Use Case:

-   AWS EMR which is big data processing tool where you can process different data.
    -   Here, we can mix and match the machine types and cost in instance fleet.
