## Q. What is a side-car container?

-   Side car container is a container  that helps the application running in the pod to perform some commont task 
    -   We can have a side-car container whose purpose is to send logs to centralized logging system.

-   This mechanism makes sure that a common usecase accross our application can be separated.
    -   Makes our deployment smooth
    -   It will also remove certain complexities when building application.
    -   Make application smoother to deploy, scale and manage in production
