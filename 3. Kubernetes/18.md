# A DevOps Engineer tainted a node as "Noschedule". Can you still schedule a pod?


-   Yes, using tolerations


## Apply Taint on a node
-   kubectl taint nodes node1 key=app:NoSchedule

## Apply toleration to a pod

```bash
apiVersion: v1
kind: Pod
metadata:
    name: myapp
spec:
    tolerations:
        -   key: "app"
            operator: "Equal"
            value: "true"
            effect: "NoSchedule"
    containers:
        -   name: nginx
            image: nginx
```
